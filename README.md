# 🛒 Automated E-Commerce Price & Competitor Tracker

A Python-based web scraping and data processing tool designed to monitor product pricing, inventory availability, and customer ratings from e-commerce platforms. Built to help online store owners make data-driven pricing and strategy decisions.

## 🚀 Features
- **Automated Data Extraction**: Scrapes product titles, prices, ratings, and stock status.
- **Data Cleaning & Normalization**: Strips unwanted character artifacts and normalizes text outputs.
- **Excel-Friendly Export**: Uses `utf-8-sig` encoding to guarantee clean rendering of currency symbols in Microsoft Excel.

## 🛠️ Tech Stack
- **Language**: Python 3.x
- **Libraries**: `requests`, `beautifulsoup4`, `pandas`
